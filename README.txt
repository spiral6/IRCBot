Extract the swt and swt_debug jars from the swtzip.
Add these to your Java classpath (look up how to do this)
Run the initGUI from the /src directory
Your OAUTH code may be found by Googling "Twitch Auth Key", this allows Twitch to authenticate the program's connection

NOTE: Keybinds can be changed in the MyThread class by changing the If parameters or what they do.
